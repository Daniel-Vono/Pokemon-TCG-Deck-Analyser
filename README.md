# Pokemon-TCG-Deck-Analyser
This is A website where you paste your Pokemon TCG decklist and it outputs probabilities and information on your deck.
## Currently Supported Information:
- Deck Composition
- Probabilitiy of openning with n basics
- Probability of n mulligans occuring
- Probability of prizing n copies of a card when m are played in the deck
- Probability of a forced start
